PhotoStack has moved to photostack.app!

PhotoStack has been updated, and the main editor is now available at photostack.app instead of edit.photostack.app. If you have PhotoStack installed, you will need to open the new site and install it again through your web browser.

If you have the PhotoStack Microsoft Store application installed on your Windows PC, it should be updated to new version soon, or you can install the new version through a web browser now. The PhotoStack Android application is no longer supported.

Watermarks cannot be automatically migrated to the new PhotoStack. If you wan to move your watermarks, click the export button below and then re-import them from the new watermarks page.
